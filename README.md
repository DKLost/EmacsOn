# EmacsOnWin

Make CapsLock a Ctrl (You can use shift+CaspLock to toggle CapsLock mode)

Allows you to use emacs-like key bindings on Windows environment.



---



## Keybindings

| Keybinding     | Function(Emacs Lisp)     |
| -------------- | ------------------------ |
| C-Space        | set-mark-command         |
| C-@            | set-mark-command         |
| C-x C-f        | find-file                |
| C-x C-s        | save-buffer              |
| C-x C-c        | kill-emacs               |
| C-d            | delete-char              |
| C-x C-/        | redo                     |
| C-k            | kill-forward             |
| C-x C-k        | kill-backward            |
| C-g            | quit                     |
| C-j            | newline-and-indent       |
| C-m            | newline                  |
| C-i            | indent-for-tab-command   |
| C-s            | isearch-forward          |
| C-r            | isearch-backward         |
| C-w            | kill-region              |
| M-w            | kill-ring-save           |
| C-y            | yank                     |
| C-/            | undo                     |
| C-a            | move-beginning-of-line   |
| C-e            | move-end-of-line         |
| C-p            | previous-line            |
| C-n            | next-line                |
| C-f            | forward-char             |
| C-b            | backward-char            |
| C-v            | scroll-down              |
| M-v            | scroll-up                |
| C-o            | open-line                |
| CapsLock       | Ctrl                     |
| Shift+CapsLock | CapsLcok                 |
| Win+CapsLock   | suspend/unsuspend script |
